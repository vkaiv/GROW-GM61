# GROW-barcode-rpi

Small program that shows how to interface Grow GM61 UART barcode reader with Raspberry Pi

## Description
Should probably work with other Grow barcode readers as well.

## Getting Started

### Dependencies

* WiringPi

### Usage
Compile with

```
gcc gm61.c -o gm61 -lwiringPi
```
